Mass Remove and Delete Data Odoo Apps

Clean Data Records Odoo App helps users to clean or delete unwanted data using wizard on single click of button. Users can easily clear data by clicking on checkbox like Sales & All Transfers, Purchase & All Transfers, Only Transfers, Project, Tasks & Timesheets, Only Tasks & Timesheets, Customers & Vendors, BOM & Manufacturing Orders, Only Manufacturing Orders, All Invoicing, Payments & Journal Entries, Only Journal Entries, Chart of Accounts & All Accounting Data. Users can also clean all above objects using a single click by checking the "All Data" checkbox.

Features

Easy to Clear Data or Records

User can easily clear unwanted data or record by using clear data wizard.

Clear All Data using Single Click

User can clear all object data by clicking on 'All Data' checkbox.

Activate the Developer Mode

You have to activate the developer mode to access the "Clear Data" Menu.

Clear Data Menu

After activate the developer mode you can see 'Clear Data' menu if you tick on that one wizard will be open.
